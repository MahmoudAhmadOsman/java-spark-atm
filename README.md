## 1 - Java Spark ATM

Java console based ATM project 

## Functionality

- A User will be able to
   - login using the provided ATM number & pin number
   - deposit money
   - withdraw money
   - view balance
   - view account activities
   



## Application Overview 

#### Interface

```bash

1. viewBalance
2. withdrawAmount
3. depositAmount
4. viewMiniStatement
```
